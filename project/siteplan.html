📋 PianoMastery Hub - Site Plan

1. Site Name

PianoMastery Hub

I chose this name because it clearly communicates the goal of the site: helping users achieve "mastery" of the piano through a central "hub" of educational resources.

2. Site Purpose

The purpose of PianoMastery Hub is to provide a structured, step-by-step learning path for aspiring pianists. The site will offer video tutorials, interactive sheet music, and daily practice exercises to help users progress from beginners to advanced players.

3. Target Audience Scenarios

  • "I just bought my first keyboard; how do I know which notes are which and where do I start my first lesson?"
  • "I've been playing for a few months and want to learn how to play 'Fur Elise'—where can I find a simplified tutorial?"

4. Color Scheme

This document applies the following colors from the project plan:

  • Primary Color: #003366 (Deep Blue) - Used for headers, footers, and all primary headings (H1, H2).
  • Accent Color: #FFB81C (Golden Yellow) - Used for buttons, borders, and call-to-action highlights.
  • Background Color: #F8F9FA (Light Gray) - Used for page backgrounds to maintain a clean look.

5. Typography

This document applies the following fonts from the project plan:

  • Georgia (Serif): Applied to all Headings (H1, H2, H3) for a classic, musical feel.
  • Arial (Sans-Serif): Applied to all Body Text and Paragraphs for high readability.
