Talented Keys

🎹 Online and Home Base Piano Lesons- Site Plan

1. Site Name

Talented Keys (PianoMastery Hub)

I chose this name because it clearly communicates the goal of the site: helping users achieve "mastery" of the piano through a central "hub" of educational resources.

2. Site Purpose

The purpose of Talented Keys is to provide a structured, step-by-step learning path for aspiring pianists. The site will offer video tutorials, interactive sheet music, and daily practice exercises.

3. Target Audience Scenarios

  • "I just bought my first keyboard; how do I know which notes are which?"
  • "I want to learn how to play 'Fur Elise'—where can I find a simplified tutorial?"

4. Color Scheme

  • Primary Color: #1B4332 (Forest Green)
  • Accent Color: #52B788 (Emerald Green)
  • Background: #FFFFFF (White)

5. Typography

  • Georgia (Serif): Headings (H1, H2, H3)
  • Arial (Sans-Serif): Body Text
